# Sam Chhabra - Portfolio

Welcome to the repository for my personal portfolio website.

## Overview

This portfolio is built to showcase my projects, skills, and experience in scalable systems, cloud computing, and AI technologies. It features a modern, physics-driven UI with fluid animations.

## Featured Projects
- **SkyLink:** Distributed Drone Network Simulator built in Rust.
- **Distributed Pong:** Embedded deterministic control system over CAN bus.
- **ML Demand Prediction:** Production-grade predictive forecasting model.
- **JMS Platform:** Interactive workflow interface using Three.js and React.

## Tech Stack
- **Frontend Core:** React, Vite
- **Styling:** Tailwind CSS, SCSS
- **Animations & Layout:** Framer Motion, Lenis Smooth Scroll
- **Icons & Forms:** Lucide React

## Running Locally

To run this repository locally:

```bash
# 1. Install dependencies
npm install

# 2. Start the local development server
npm run dev
```
